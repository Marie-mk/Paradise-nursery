# 🌿 Paradise Nursery Shopping Application

## Project Overview

Paradise Nursery is a dynamic React-based e-commerce web application for an online plant shop. Users can browse a curated collection of houseplants, add them to a shopping cart, and manage their selections in real time.

## Features

- 🌱 Browse plants organized into categories (Tropical, Succulents & Cacti, Air-Purifying)
- 🛒 Add plants to a shopping cart with live cart count updates
- ➕➖ Increase or decrease item quantities directly in the cart
- 🗑️ Remove individual items from the cart
- 💰 View per-item totals and overall cart total
- 🔗 Navigate between the landing page, product listing, and cart via a persistent navbar

## Tech Stack

- React – Component-based UI
- Redux Toolkit – State management for the shopping cart
- React Router DOM – Client-side routing
- Vite – Development server and bundler
- CSS3 – Custom styling with CSS variables and animations
